# SQLMap
> SQLMap是一款开源的自动化SQL注入工具，主要用于检测和利用Web应用程序中的SQL注入漏洞，支持多种数据库管理系统（如MySQL、Oracle、PostgreSQL等）

 

 ### 使用
 直接去 [官网](https://sqlmap.org/) 下载 `ZIP` 包。

 ```python
 python sqlmap.py -u "http://www.example.com/login.php?id="
 ```

 具体教程看 https://blog.csdn.net/shanguicsdn111/article/details/142333841
